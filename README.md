# GYANI Science Search

Local science search engine with 10,000+ generated topic records, smarter relevance ranking, and external learning links.

## Run

Open PowerShell in this folder and run:

```powershell
start .\index.html
```

## Notes

- The large dataset is generated locally in `science-data.js`.
- Links point to external educational/search sources like Wikipedia, NASA, ESA, NOAA, USGS, Khan Academy, MIT OCW, LibreTexts, Britannica, and MathWorld.
- This is a seeded science knowledge engine, not a fully verified scientific database.
